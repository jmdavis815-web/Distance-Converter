# Distance Converter

A simple web application that converts distances from kilometers to miles.

## Features

- Clean and intuitive user interface
- Real-time conversion from kilometers to miles
- Input validation with error handling
- Responsive design using Bootstrap 5

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5.3.8

## How to Use

1. Open `index.html` in your web browser
2. Enter a distance value in kilometers
3. Click the "Convert to Miles" button
4. The converted value in miles will be displayed below

## Conversion Formula

The application uses the standard conversion formula:
```
Miles = Kilometers / 1.60934
```

## Files

- `index.html` - Main HTML structure
- `app.js` - JavaScript conversion logic
- `style.css` - Custom styling

## Installation

No installation required! Simply clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/westfallt13-dot/Distance-Converter.git
cd Distance-Converter
```

Then open `index.html` in your preferred web browser.
